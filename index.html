Projetos Finalizados

Interface para Auditoria de Estoque

Nesta auditoria, realizamos a validação dos produtos armazenados em determinados locais no estoque, no qual chamamos de "Endereços". Desenvoli esta interface com o intuíto de facilitar a comparação da quantidade dos produtos armazenados e a quantidade de produtos que foram validados. Além disso, também permite registrar inconformidades e observações durante o processo de auditoria.
Está interface foi desenvolvida em uma planilha Google, foi utilizado, HTML, CSS e Google Apps Script.

Interface para Auditoria de Pedidos

Utilizando o mesmo conceito da planilha de auditoria de estoque, criei uma interface personalizada para a auditoria de pedidos. Essa interface permite uma análise mais rápida e eficiente dos dados, além de gerar relatórios automáticos que ajudam na tomada de decisões.
